# Portfolio Profesional | Enmanuel Fuenmayor

Este repositorio contiene mi portafolio personal, diseñado para mostrar mis habilidades, proyectos y experiencia como Desarrollador Web Full Stack. El objetivo es ofrecer una experiencia digital fluida, moderna y profesional.

## Tecnologías Utilizadas

El proyecto ha sido construido utilizando herramientas modernas que garantizan un alto rendimiento y una excelente experiencia de usuario:

- **Frontend**: React con TypeScript para un desarrollo robusto y mantenible.
- **Estilos**: Tailwind CSS para un diseño personalizado, responsivo y eficiente.
- **Animaciones**: Framer Motion, aportando interactividad y dinamismo a la interfaz.
- **Iconografía**: Lucide React para un conjunto de iconos limpio y consistente.
- **Herramientas de Construcción**: Vite, asegurando tiempos de carga rápidos y un flujo de trabajo ágil.

## Características Principales

- **Diseño Adaptable**: Interfaz optimizada para dispositivos móviles, tablets y escritorio.
- **Modos Claro y Oscuro**: Implementación nativa de temas con alta legibilidad en ambas variantes.
- **Secciones Dinámicas**: Incluye información sobre mí, mis habilidades técnicas, un carrusel de proyectos y una sección de contacto.
- **Formulario de Contacto Funcional**: Integración con Formspree para la recepción directa de consultas vía correo electrónico.
- **Accesibilidad y SEO**: Uso de HTML semántico y mejores prácticas para garantizar que el contenido sea accesible y fácil de indexar.
